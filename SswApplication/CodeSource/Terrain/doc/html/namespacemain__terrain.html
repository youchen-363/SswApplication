Parameters
[in]Inputsare defined in the files in the "inputs" directory
  • terrain.csv that contains
    – N_x: number of horizontal points
    – z_step: step along the vertical in m
    – z_max_relief: max altitude of the relief in m
    – iterations: number of scales in the multiscale procedural generation
    – width: number of pts defining the width of the relief (Triangle) in pts
    – center: number of pts along x at which is the center of the relief (Triangle) in pts
[out]z_relief(N_x)-array. Contains altitudes of the relief. Scaled between 0 and z_max_relief
